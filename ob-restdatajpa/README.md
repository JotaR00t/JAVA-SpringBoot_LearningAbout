## Spring Boot
Proyecto Spring Boot con las dependencias / starters:
* H2
* Spring Data JPA
* Spring Web
* Spring Boot Dev tools

Aplicación API REST con acceso a BBDD H2 para persistir la información.

El acceso se puede realizar desde Postman o Navegador.


## Entidad book

1. Book
2. BookRepository
3. BookController
